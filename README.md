# Installation
1. Create and activate a Python virtual environment.
```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies.
```bash
pip install -r requirements.txt
```

3. Install Mossie package.
```bash
pip install -e .
```

# Usage
1. Run the start script.
```bash
./bin/mossie
```
